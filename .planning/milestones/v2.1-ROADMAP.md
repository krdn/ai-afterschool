# Milestone v2.1: Parent Counseling Management

**Status:** ✅ SHIPPED 2026-02-05
**Phases:** 16-22
**Total Plans:** 30

## Overview

선생님 중심의 학부모 상담 예약/기록 시스템 구축으로 체계적인 상담 관리 및 통계 제공. 학부모 계정 없이 선생님이 모든 상담 관리를 담당하며, 외부 알림 없이 시스템 내부에만 기록하는 운영 방식.

## Phases

### Phase 16: Parent & Reservation Database Schema

**Goal**: 학부모 정보 및 상담 예약 데이터 모델 구축
**Depends on**: v2.0 Teacher Management
**Plans**: 1 plan

Plans:
- [x] 16-01: Parent 및 ParentCounselingReservation Prisma 모델 정의 및 DB 마이그레이션

**Details:**
- Parent 모델: name, phone, ParentRelation enum (FATHER/MOTHER/GRANDFATHER/GRANDMOTHER/OTHER), relationOther, isPrimary
- ParentCounselingReservation 모델: scheduledAt, topic, status (SCHEDULED/COMPLETED/CANCELLED/NO_SHOW), notes
- Student FK 간접 격리로 기존 RBAC Extension 재사용
- 주 연락처 이중 저장 (Student.primaryParentId + Parent.isPrimary)
