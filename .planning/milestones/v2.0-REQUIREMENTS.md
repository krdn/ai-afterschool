# Requirements: AI AfterSchool v2.0

**Defined:** 2026-01-30
**Core Value:** 학생 정보 통합 관리를 기반으로 AI 성향 분석 및 맞춤형 학습/진로 제안 제공

## v2.0 Requirements

다중 선생님 지원, 팀 기반 접근 제어, 선생님-학생 궁합 분석, AI 기반 최적 배정 시스템.

### 선생님 관리 (TEACH)

- [x] **TEACH-01**: 선생님 기본 정보 관리 (이름, 이메일, 연락처, 사진, 소속 팀)
- [x] **TEACH-02**: 선생님 계층 구조 (원장 > 팀장 > 매니저 > 선생님)
- [x] **TEACH-03**: 팀 단위 데이터 접근 제어 (원장: 전체, 팀장: 소속 팀만)
- [x] **TEACH-04**: 선생님 성향 분석 (MBTI, 사주, 성명학, 관상, 손금)
- [x] **TEACH-05**: 선생님 목록 조회 및 검색
- [x] **TEACH-06**: 선생님 상세 정보 조회

### 매칭 시스템 (MATCH)

- [x] **MATCH-01**: 선생님-학생 궁합 분석 (AI 기반)
- [x] **MATCH-02**: 학생 배정 기능 (선생님에게 학생 할당)
- [x] **MATCH-03**: AI 자동 배정 제안 (궁합 + 부하 분산)
- [x] **MATCH-04**: 학생별 선생님 추천 (맞춤 순위)

### 성과 분석 (PERF)

- [x] **PERF-01**: 선생님 성과 분석 (학생 성과 추적)
- [x] **PERF-02**: 팀 구성 분석 (성향 조합, 전문성 커버리지)
- [x] **PERF-03**: 공정성 메트릭 (Disparity Index, ABROCA)

### AI 설정 (AI)

- [x] **AI-01**: 다중 LLM 설정 (Ollama 로컬, Gemini, ChatGPT, Claude)
- [x] **AI-02**: Admin 설정 기능 (원장 전용)
