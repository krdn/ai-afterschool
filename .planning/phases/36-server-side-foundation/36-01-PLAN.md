  `src/lib/chat/mention-types.ts` 파일을 새로 생성한다. 다음 타입들을 정의:

  1. **MentionType**: `'student' | 'teacher' | 'team'` 리터럴 유니온
  2. **MentionItem**: 클라이언트에서 서버로 전송되는 멘션 튜플
     - `{ type: MentionType; id: string }`
  3. **MentionedEntity**: ChatMessage.mentionedEntities JSON에 저장되는 메타데이터
     - `{ id: string; type: MentionType; displayName: string; accessDenied?: boolean }`
     - accessDenied는 RBAC 실패한 멘션 추적용 (선택적 필드)
  4. **ResolvedMention**: mention-resolver가 반환하는 해결된 멘션 데이터
     - `{ item: MentionItem; displayName: string; contextData: string; accessDenied: boolean; deniedReason?: string }`
     - contextData: context-builder가 system prompt에 주입할 엔티티 요약 텍스트
  5. **MentionResolutionResult**: mention-resolver의 전체 반환 타입
     - `{ resolved: ResolvedMention[]; metadata: MentionedEntity[]; accessDeniedMessages: string[] }`
     - accessDeniedMessages: "○○○님은 접근 권한이 없어 제외되었습니다" 형태의 알림 메시지 배열

  파일 상단에 `// 멘션 시스템 공유 타입 — Phase 36~40에서 사용` 주석 추가.
