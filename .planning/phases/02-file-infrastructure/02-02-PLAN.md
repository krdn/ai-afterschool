<task type="auto">
  <name>Task 2: Add Cloudinary helper and signed upload endpoint</name>
  <files>
    package.json
    package-lock.json
    src/lib/cloudinary.ts
    src/app/api/cloudinary/sign/route.ts
  </files>
  <action>
    Install `cloudinary` and create `src/lib/cloudinary.ts` with:
    - Cloudinary v2 config from env vars.
    - Helper to create signature for params (timestamp, folder, transformation).
    - Helper to build resized URL with square crop (same rule for all types).
    Create POST `/api/cloudinary/sign` route handler:
    - Verify session and reject unauthenticated requests.
    - Accept JSON body with image type (profile/face/palm) plus optional `studentId` or `draftId`.
    - If `studentId` is present, build folder `students/{studentId}/{type}`.
    - If `studentId` is missing, require or generate `draftId` and build folder `students/drafts/{draftId}/{type}`.
    - Return signature, timestamp, apiKey, cloudName, folder, and transformation params.
  </action>
  <verify>`npm run lint`</verify>
  <done>Server can issue signed upload params and build resized URLs consistently.</done>
</task>
