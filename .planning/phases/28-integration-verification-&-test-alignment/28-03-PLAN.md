### Task 2: Fix Timing and Async Issues

**Action:** Add proper waits for dynamic content and API calls.

**Common patterns to fix:**

```typescript
// BEFORE (racy)
await page.click('[data-tab="analysis"]');
await expect(page.locator('[data-testid="saju-panel"]')).toBeVisible();

// AFTER (proper wait)
await page.click('[data-tab="analysis"]');
await page.waitForSelector('[data-testid="saju-panel"]', { state: 'attached', timeout: 5000 });
await expect(page.locator('[data-testid="saju-panel"]')).toBeVisible();

// For API-driven content
await page.click('[data-testid="run-analysis-button"]');
await page.waitForURL(/loading/);
await page.waitForSelector('[data-testid="analysis-results"]', { state: 'visible', timeout: 10000 });
```

**Files likely needing updates:**
- `tests/e2e/analysis.spec.ts` (AI analysis waits)
- `tests/e2e/student.spec.ts` (Tab content loading)
- `tests/e2e/admin.spec.ts` (System status updates)
