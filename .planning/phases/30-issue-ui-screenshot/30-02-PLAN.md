    Props 인터페이스:
    ```typescript

      imageUrl: string;                    // 캡처된 이미지 URL (Blob URL 또는 업로드된 URL)
      isUploaded: boolean;                 // 업로드 완료 여부
      onRetake: () => void;                // 다시 캡처
      onConfirm: () => void;               // 확인 (업로드 또는 완료)
      onCancel: () => void;                // 취소
      isUploading?: boolean;               // 업로드 중 여부
    }
    ```
    
    UI 구성:
    1. 이미지 미리보기 영역 (max-height: 300px, object-contain)

       - "취소" (X 아이콘) - variant="ghost"
       - "확인" 또는 "업로드" (Check 아이콘) - variant="default"

    - 업로드 완료: [다시 캡처] [확인]
