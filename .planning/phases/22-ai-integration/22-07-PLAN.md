Phase 22: AI Integration 전체 기능

  <how-to-verify>
**사전 조건:**
- 개발 서버 실행: `npm run dev`
- 선생님 계정으로 로그인
- 성향 분석이 완료된 학생 1명 필요

**테스트 시나리오:**

1. **AI-01: 성향 분석 정보 표시**
   - 학생 상세 페이지로 이동
   - 상담 섹션에서 "상담 기록" 버튼 클릭
   - 상담 폼 우측에 "AI 지원" 버튼 확인
   - 버튼 클릭 시 사이드 패널 열림 확인
   - 성향 요약 카드에 학생 성향 정보 표시 확인
   - (성향 분석 없는 학생: "분석이 아직 완료되지 않았습니다" 메시지 확인)

2. **AI-02: 궁합 점수 참조**
   - 사이드 패널에서 궁합 점수 카드 확인
   - 점수(숫자)와 해석 텍스트("좋은 궁합" 등) 함께 표시 확인
   - "세부 항목 보기" 클릭 시 MBTI, 사주 등 세부 점수 펼쳐짐 확인
   - (궁합 점수 없는 경우: "지금 계산하기" 버튼 표시 확인)

3. **AI-03: AI 요약 생성**
   - 상담 내용 입력 (10자 이상)
   - 사이드 패널의 "AI 요약 생성" 버튼 클릭
   - 로딩 상태 표시 확인
   - 요약 생성 완료 후 텍스트 표시 확인
   - "요약 적용" 버튼 클릭
   - 상담 내용 필드에 AI 요약이 반영됨 확인
   - 상담 저장 후 DB에 aiSummary 필드 저장 확인

4. **반응형 테스트**
   - 브라우저 창을 모바일 크기로 조정 (375px)
   - 사이드 패널이 전체 너비로 표시됨 확인

**예상 결과:**
- AI 지원 버튼 클릭 시 패널 열림
- 성향/궁합 정보가 순서대로 표시
- AI 요약 생성 및 적용 정상 동작
- 모바일에서 패널이 전체 너비 사용
