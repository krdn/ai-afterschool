   - 매개변수: personality (UnifiedPersonalityData), studentName

```typescript
interface CounselingSummaryPromptParams {

    sessionDate: Date
    type: string
  }>
  studentName: string
}

interface PersonalitySummaryPromptParams {
  personality: UnifiedPersonalityData
  studentName: string
}
```
