# Plan 25-01: Student Page UI Enhancement (이미지, 검색, 삭제)

## Overview

학생 페이지의 UI 정합성을 개선합니다. 이미지 alt 속성, 검색 결과 표시, 삭제 후 리다이렉트를 수정합니다.

## Tasks

### Task 1: 이미지 alt 속성 정합성 확보 (STU-02, UTL-02)

**파일:** `src/components/students/student-image-uploader.tsx`

1. `studentName` prop 추가
2. alt 속성을 학생 이름 포함 형식으로 변경:
   ```tsx
   // Before
   alt={`${label} 미리보기`}

   // After
   alt={`${studentName}의 ${label} 사진`}
   ```

3. `CldImage` 속성 확인:
   - `width`, `height` 설정되어 있는지 확인
   - `sizes` 속성 적절한지 확인

### Task 2: 학생 목록 이미지 lazy loading (UTL-01)

**파일:** `src/components/students/columns.tsx`

1. 이미지 컬럼이 있는지 확인
2. 이미지가 있으면 `loading="lazy"` 속성 추가

### Task 3: 검색 결과 빈 상태 data-testid 추가 (STU-03)

**파일:** `src/components/students/student-table.tsx`

1. 빈 결과 메시지에 `data-testid="empty-search-result"` 추가:
   ```tsx
   <TableCell colSpan={columns.length} className="h-24 text-center" data-testid="empty-search-result">
     검색 결과가 없어요.
   </TableCell>
   ```

### Task 4: 학생 삭제 후 리다이렉트 확인 (STU-04)

**파일:** `src/components/students/student-detail-actions.tsx`

1. 삭제 로직 확인:
   - `router.push('/students')`가 있는지 확인
   - toast 알림 후 리다이렉트되는지 확인

2. 없으면 추가:
   ```tsx
   toast.success('학생이 삭제되었습니다.')
   router.push('/students')
   ```
