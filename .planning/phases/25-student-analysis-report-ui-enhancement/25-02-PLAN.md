## Overview

분석 탭 내에서 사주/관상/손금/MBTI를 서브탭으로 분리하고, AI 분석 실패 시 에러 메시지와 재시도 버튼을 통일합니다.

## Tasks

### Task 1: AnalysisTab 서브탭 구조 변경 (ANL-02)

**파일:** `src/components/students/tabs/analysis-tab.tsx`

1. shadcn/ui Tabs 컴포넌트 중첩 구조로 변경:
   ```tsx
   import { Tabs, TabsList, TabsTrigger, TabsContent } from "@/components/ui/tabs"
   import { SajuAnalysisPanel } from "@/components/students/saju-analysis-panel"
   import { FaceAnalysisPanel } from "@/components/students/face-analysis-panel"
   import { PalmAnalysisPanel } from "@/components/students/palm-analysis-panel"
   import { MbtiAnalysisPanel } from "@/components/students/mbti-analysis-panel"

   export default function AnalysisTab({ studentId, studentName }: { studentId: string; studentName: string }) {
     const [subTab, setSubTab] = useState("saju")

     return (
       <Tabs value={subTab} onValueChange={setSubTab} data-testid="analysis-sub-tabs">
         <TabsList className="grid w-full grid-cols-4">
           <TabsTrigger value="saju">사주</TabsTrigger>
           <TabsTrigger value="face">관상</TabsTrigger>
           <TabsTrigger value="palm">손금</TabsTrigger>
           <TabsTrigger value="mbti">MBTI</TabsTrigger>
         </TabsList>
         <TabsContent value="saju">
           <SajuAnalysisPanel student={{ id: studentId, name: studentName, ... }} />
         </TabsContent>
         <TabsContent value="face">
           <FaceAnalysisPanel studentId={studentId} />
         </TabsContent>
         <TabsContent value="palm">
           <PalmAnalysisPanel studentId={studentId} />
         </TabsContent>
         <TabsContent value="mbti">
           <MbtiAnalysisPanel studentId={studentId} />
         </TabsContent>
       </Tabs>
     )
   }
   ```

### Task 2: 에러 메시지 통일 (ANL-03)

**파일:**

1. 공통 에러 메시지 형식 적용:
   ```tsx
   // 사주
   `사주 분석에 실패했습니다. (원인: ${error}) 다시 시도해주세요.`

   // 관상/손금
   `이미지 분석에 실패했습니다. (원인: ${error}) 다시 시도해주세요.`

   // MBTI
   `MBTI 분석에 실패했습니다. (원인: ${error}) 다시 시도해주세요.`
   ```

2. 재시도 버튼 스타일 통일:
   ```tsx
   <Button onClick={onRetry} disabled={isRetrying} variant="outline">
     {isRetrying ? (
       <>
         <Loader2 className="w-4 h-4 animate-spin" />
         재시도 중...
       </>
     ) : (
       <>
         <RefreshCw className="w-4 h-4" />
         다시 시도
       </>
     )}
   </Button>
   ```

3. data-testid 추가:
   - 에러 상태: `data-testid="analysis-error"`
   - 재시도 버튼: `data-testid="retry-button"`
