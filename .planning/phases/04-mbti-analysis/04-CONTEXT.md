# Phase 4: MBTI Analysis - Context

**Gathered:** 2026-01-29
**Status:** Ready for planning

<domain>
## Phase Boundary

설문 기반 MBTI 성향 판정 시스템 — 선생님이 학생을 대신해 60문항 설문을 진행하고, MBTI 유형(E/I, S/N, T/F, J/P)과 차원별 백분율을 계산하여 학생 프로필에 저장합니다. 설문 중간 저장, 재수정, 상세 결과 표시를 지원합니다.

</domain>

<decisions>
## Implementation Decisions

### 설문 진행 방식
- **한 페이지에 전체 문항** 표시 (스크롤하며 응답)
- **중간 저장 기능 필수** (DB에 저장)
  - 제출할 때까지 보관
  - 중단 후 나중에 이어서 진행 가능
- **진행률 표시**: 퍼센트 바 + 숫자 (예: "45/60 (75%)")
- **응답 검증**: 미응답 문항을 빨갛게 강조 표시하고 제출 불가
- **이탈 경고**: 페이지 이동 시 "작성 중인 내용이 저장됩니다" 확인 창 표시
- **제출 후 수정 가능**: 제출한 MBTI 설문을 다시 수정할 수 있음

### 문항 구성 및 출처
- **표준 MBTI 문항 사용** (MBTI Form M 등 공식 문항)
- **총 60문항** (4개 차원 × 15문항씩)
- **한국어 번역**: MBTI 한국어판 공식 번역본 사용
- **문항 순서**: 차원별 그룹화
  - E/I 문항 15개 → S/N 15개 → T/F 15개 → J/P 15개

### 결과 표시 방식
- **MBTI 유형 (ENFP 등)** 표시
- **차원별 백분율** (예: E 70% / I 30%)
  - 프로그레스 바로 시각화: E [████████░░] 70%
- **유형별 설명** (예: "ENFP는 재기 발랄한 활동가입니다")
- **강점/약점**: 유형별 전체 설명 (강점 3-5개, 약점 3-5개)
- **추가 내용 모두 표시**:
  - 적합한 학습 방식 (예: "시각적 학습이 효과적입니다")
  - 추천 직업/학과 (예: "예술, 마케팅, 교육 분야 추천")
  - 유명인 사례 (예: "같은 MBTI 유형: 스티브 잡스, BTS RM")

### 답변 입력 UI
- **5점 척도 라디오 버튼**
  - 라벨: 숫자 (1, 2, 3, 4, 5)
- **키보드 단축키 지원**: 키보드 1~5 키로 빠르게 응답 가능
- **문항 설명 표시**: 각 문항 아래에 예시/설명 표시 (예: "예: 친구들과 시간 보내는 것")

### Claude's Discretion
- 임시 저장 데이터의 정확한 스키마 구조
- 진행률 바의 시각적 디자인
- 결과 페이지의 레이아웃 구성
- 유명인 사례의 구체적인 선정 기준

</decisions>

<specifics>
## Specific Ideas

- 설문은 "한 번에 쭉 읽고 답하는" 방식이 아니라, 선생님이 학생에게 질문하고 답을 받아 입력하는 방식이므로 **중간 저장이 매우 중요**
- 차원별 그룹화를 통해 "지금은 E/I에 대한 질문들입니다" 같은 구분이 명확하게 보이도록 해야 함
- 결과는 학부모 상담 시 활용할 수 있도록 **시각적으로 깔끔하고 전문적**으로 보여야 함

</specifics>
