**src/lib/analysis/mbti-scoring.ts:**

Create scoring engine with:

1. `calculateProgress(responses: Record<string, number>, totalQuestions: number)`:
   - Returns { answeredCount, totalQuestions, percentage }
   - Only counts keys present in responses object

2. `scoreMbti(responses: Record<string, number>, questions: Question[])`:
   - Input: responses object { "1": 3, "2": 5, ... }, questions array with dimension/pole info
   - Process: Sum scores per pole using question metadata
   - Calculate percentages per dimension
   - Determine MBTI type (higher score wins each dimension)
   - Return: { scores, mbtiType, percentages }

Scoring logic:
```typescript
// For each response, add score to the pole that question measures
// Score 1-5 scale: 1 = strong disagree, 5 = strong agree
// If question.pole === 'E' and response === 5, add 5 to E score
for (const question of questions) {
  const response = responses[question.id.toString()]
  if (response !== undefined) {
    scores[question.pole.toLowerCase()] += response
  }
}
```
