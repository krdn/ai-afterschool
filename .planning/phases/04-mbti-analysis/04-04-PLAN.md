<task type="checkpoint:human-verify" gate="blocking">
  <name>Verify MBTI survey and results flow</name>
  <what-built>
MBTI 설문 시스템:
- 60문항 설문 페이지 (한 페이지 스크롤)
- 자동 저장 (2초 debounce)
- 키보드 단축키 (1-5)
- 진행률 표시
- 결과 계산 및 저장
- 학생 상세 페이지 결과 패널
  </what-built>
  <how-to-verify>
**사전 조건:** `npm run dev`로 개발 서버 실행

**테스트 1: 설문 시작**
1. 학생 상세 페이지 접속 (/students/{id})
2. MBTI 패널에서 "MBTI 검사 시작" 버튼 클릭
3. 설문 페이지로 이동 확인
4. 60문항이 차원별로 그룹화되어 표시되는지 확인

**테스트 2: 자동 저장**
1. 몇 개 문항에 응답
2. 3초 정도 대기
3. 페이지 새로고침 (F5)
4. 응답이 유지되는지 확인

**테스트 3: 키보드 단축키**
1. 스크롤하여 문항에 포커스
2. 키보드 1-5 키로 응답
3. 응답이 선택되고 다음 문항으로 스크롤되는지 확인

**테스트 4: 진행률**
1. 응답할 때마다 진행률 바가 업데이트되는지 확인
2. "45/60 (75%)" 형식으로 표시되는지 확인

**테스트 5: 제출 검증**
1. 일부 문항을 비워두고 "설문 제출" 클릭
2. 미응답 문항으로 스크롤되고 빨간 테두리가 표시되는지 확인
3. 모든 문항에 응답 후 다시 제출
4. 학생 상세 페이지로 리다이렉트되는지 확인

**테스트 6: 결과 표시**
1. 학생 상세 페이지에서 MBTI 패널 확인
2. MBTI 유형 (예: ENFP) 배지 표시
3. 4개 차원의 백분율 바 표시
4. 강점/약점, 학습 방식, 추천 직업 표시
5. 유명인 사례 표시
6. 면책 조항 텍스트 표시

**테스트 7: 수정**
1. MBTI 패널의 "수정" 버튼 클릭
2. 설문 페이지에서 기존 응답이 로드되는지 확인
3. 일부 응답 변경 후 제출
4. 결과가 업데이트되는지 확인
  </how-to-verify>
