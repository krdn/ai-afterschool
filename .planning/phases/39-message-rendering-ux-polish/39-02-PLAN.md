`renderUserContent` 함수 구현 (컴포넌트 외부 또는 내부 useMemo):
```typescript
function renderUserContent(content: string, entities?: MentionedEntity[] | null) {
  // mentionedEntities가 없으면 plain text
  if (!entities || entities.length === 0) {
    return content
  }

  const segments = parseMentionChips(content, entities)

  return segments.map((seg, i) => {
    if (seg.kind === "text") return <span key={i}>{seg.text}</span>
    return <MentionTag key={`${seg.entity.id}-${i}`} entity={seg.entity} />
  })
}
```

import 추가:
```typescript

```

**주의사항:**
- `<p>` → `<div>` 변환: `<p>` 안에 `<span>` + Radix `<Popover>`를 넣으면 HTML validation 에러. `<div>`로 변경하되 동일 스타일 유지.

- assistant 메시지는 MarkdownRenderer를 사용하므로 파싱하지 않음
- `renderUserContent`를 별도 함수로 추출하여 컴포넌트 body를 깔끔하게 유지
