  <name>Task 1: Fairness Metrics 순수 함수들 구현</name>
  <files>src/lib/analysis/fairness-metrics.ts</files>
  <action>
  1. src/lib/analysis/fairness-metrics.ts 생성

  2. 타입 정의:
     - FairnessInput: { compatibilities: CompatibilityResult[], groupAttribute: string }
     - DisparityIndexOutput: { index: number, groupMeans: Record<string, number>, maxDiff: number }
     - ABROCAOutput: { score: number, curves: Record<string, number[]> }
     - DistributionBalanceOutput: { balance: number, teacherDistribution: Record<string, number> }

  3. calculateDisparityIndex 함수:
     a) groupAttribute로 그룹화 (예: student.school, student.grade)
     b) 각 그룹의 평균 궁합 점수 계산
     c) 최대-최소 차이를 전체 평균으로 정규화
     d) Disparity Index = (max_score - min_score) / overall_mean
     e) 결과 반환: { index, groupMeans, maxDiff }
     f) 인덱스 < 0.1이면 "공정함", 0.1-0.3이면 "주의 필요", > 0.3이면 "편향 있음"

  4. calculateABROCA 함수:
     a) 각 그룹의 궁합 점수 분포를 누적 함수로 변환
     b) 그룹 간 누적 함수의 면적 차이 계산 (Approximate Between ROC Curves)
     c) CDF 계산: score -> rank / total
     d) 그룹 간 CDF 차이의 절대값 적분 (합계)
     e) 결과 반환: { score, curves }
     f) 점수 < 0.05이면 "분포 유사", > 0.1이면 "분포 차이 큼"

  5. calculateDistributionBalance 함수:
     a) 선생님별 학생 수 분포 계산
     b) 엔트로피 계산: -sum(p_i * log(p_i)), p_i = count_i / total
     c) 최대 엔트로피로 정규화 (균등 분포 기준)
     d) Balance Score = entropy / max_entropy
     e) 결과 반환: { balance, teacherDistribution }
     f) 점수 > 0.8이면 "균형 잡힘", 0.5-0.8이면 "보통", < 0.5이면 "불균형"

  6. calculateOverallFairness 함수 (통합):
     - 세 가지 메트릭 모두 계산
     - 가중 평균 (Disparity 40%, ABROCA 30%, Balance 30%)
     - 전체 공정성 점수 반환 (0-100)
  </action>
