# Plan 14-09: Student Learning Tab UI

## Objective

학생 상세 페이지의 학습 현황 탭에서 성적 관리 기능을 제공한다.

## Requirements

- Phase 14-01에서 생성된 GradeHistory 모델 활용
- 학생별 성적 추이 차트 시각화
- 성적 CRUD (추가/조회/삭제) 기능

## Tasks

### Task 1: Grade Server Actions

**파일:** `src/lib/actions/grade.ts`

- `addGrade()`: 성적 추가 (Zod 검증, teacherId 연결)
- `getGrades()`: 학생별 성적 목록 조회
- `deleteGrade()`: 성적 삭제

### Task 2: Learning Tab Component

**파일:** `src/components/students/tabs/learning-tab.tsx`

- 성적 추이 차트 (Recharts LineChart)
- 성적 목록 테이블
- 성적 추가 Dialog 폼
- 성적 삭제 기능

### Task 3: Page Integration

**파일:** `src/app/(dashboard)/students/[id]/page.tsx`

- LearningTab 컴포넌트 import 및 통합

## Technical Decisions

| Decision | Rationale |
|----------|-----------|
| Recharts LineChart | 기존 프로젝트에서 사용 중, 일관성 유지 |
| Dialog 기반 폼 | 페이지 이동 없이 빠른 입력 |
| 점수별 색상 구분 | 90+:녹색, 80+:파랑, 70+:노랑, 70-:빨강 |

## Success Criteria

- [ ] 성적 추가/조회/삭제 동작
- [ ] 성적 추이 차트 표시
- [ ] 빌드 오류 없음
