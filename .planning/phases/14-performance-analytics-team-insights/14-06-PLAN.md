    - 입력: 팀 ID
    - 출력: TeamCompositionAnalysis
    
    **분석 항목:**
    1. **MBTI 분포:**
       - 16가지 유형별 카운트
       - E/I, S/N, T/F, J/P 축별 비율
       - 가장 흔한/희귀한 유형 식별
    
    2. **학습 스타일 분포 (VARK):**
       - Visual, Auditory, Reading, Kinesthetic 비율
       - TeacherMbtiAnalysis에서 percentages 유도
    
    3. **사주 오행 분포:**
       - 목(木), 화(火), 토(土), 금(金), 수(水) 비율
       - 주요/부족 오행 식별
    
    4. **전문성 커버리지:**
       - 담당 과목 분포 (수학, 영어, 국어, 과학 등)
       - 학년별 분포 (중1-3, 고1-3)
       - 경력 분포 (신입/중급/고급)
    
    5. **역할 분포:**
       - TEAM_LEADER, MANAGER, TEACHER 비율

    - 다양성 점수 계산 (0-100)
    - Shannon Diversity Index 사용

    **getTeamRecommendations(composition):**
    - 팀 개선 제안 생성

    - "과목 커버리지: 과학 과목이 취약합니다"
    
    **테스트:**
    - Mock 데이터로 분석 함수 테스트
