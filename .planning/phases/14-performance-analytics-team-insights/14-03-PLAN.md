  <name>calculateImprovementRate</name>
  <files>
    src/lib/analysis/grade-analytics.ts
    src/lib/analysis/__tests__/grade-analytics.test.ts
  </files>
  <behavior>
    학생의 성적 향상률을 계산합니다.
    
    **입력:**
    - gradeHistory: GradeHistory[] - 성적 이력 배열
    - baselineDate?: Date - 기준일 (기본값: 첫 성적)
    - controlVariable?: { initialLevel: 'HIGH' | 'MID' | 'LOW' }
    
    **출력:**
    - improvementRate: number - 향상률 (-100 ~ +100)
    - trend: 'UP' | 'STABLE' | 'DOWN'
    - confidence: number - 신뢰도 (0-1, 데이터 포인트 수 기반)
    
    **계산 로직:**
    1. 기준 성적(baseline) 식별 (첫 기록 또는 baselineDate 이전 마지막 기록)
    2. 최신 성적(current) 식별
    3. 향상률 = ((current - baseline) / baseline) * 100
    4. 통제 변수 적용:
       - HIGH 초기 성적: +10% 향상 = 우수 (더 높은 개선 난이도)
       - LOW 초기 성적: +30% 향상 = 우수
    5. 추세 판정:
       - improvementRate > 10%: UP
       - -10% <= improvementRate <= 10%: STABLE
       - improvementRate < -10%: DOWN
    
    **테스트 케이스:**
    1. 기본 향상: 60점 → 80점 = +33.3%
    2. 하락: 80점 → 60점 = -25%
    3. 안정: 75점 → 78점 = +4% (STABLE)
    4. 통제 변수 HIGH: 90점 → 93점 = +3.3% → 조정 후 +13.3%
    5. 통제 변수 LOW: 50점 → 65점 = +30% → 조정 후 +20%
    6. 단일 데이터: 오류 반환 (트렌드 계산 불가)
  </behavior>
