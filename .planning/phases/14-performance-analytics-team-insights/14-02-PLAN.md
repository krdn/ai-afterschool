  <name>Task 1: 선생님 담당 학생 조회 Server Actions</name>
  <files>src/lib/actions/teacher-performance.ts</files>
  <action>
    선생님의 담당 학생 목록과 통계를 조회하는 Server Actions를 구현합니다.
    
    **getTeacherStudents(teacherId, options?):**
    - 특정 선생님에게 배정된 모든 학생 조회
    - Student 테이블에서 teacherId로 조회
    - 각 학생의 최신 성적, 상담 횟수, 궁합 점수 포함
    - options: { search?, subject?, gradeRange?, sortBy? }
    
    **쿼리 구조:**
    ```typescript
    const students = await prisma.student.findMany({
      where: { teacherId, ...searchFilter },
      include: {
        gradeHistory: { orderBy: { testDate: 'desc' }, take: 1 },
        _count: { select: { counselingSessions: true } },
        compatibilityResults: {
          where: { teacherId },
          select: { overallScore: true }
        }
      }
    })
    ```
    
    **getTeacherStudentMetrics(teacherId):**
    - 총 담당 학생 수
    - 평균 성적 변화 추이
    - 총 상담 횟수
    - 평균 궁합 점수
    - 과목별 학생 분포
    
    **getStudentGradeTrend(studentId, months?):**
    - 특정 학생의 성적 변화 추이 (시간별 그래프 데이터)
    - 월별 평균 성적 계산
    
    **RBAC 및 RLS:**
    - DIRECTOR: 모든 선생님의 담당 학생 조회 가능
    - TEAM_LEADER: 자신의 팀 소속 선생님만 조회 가능
    - MANAGER/TEACHER: 자신의 담당 학생만 조회 가능
    - getRBACPrisma() 사용하여 데이터 필터링
    
    **반환 형식:**
    - { data: StudentWithMetrics[] } 또는 { error: string }
  </action>
