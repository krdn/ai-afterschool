2. **runTeacherFaceAnalysis** 함수:
   ```typescript
   export async function runTeacherFaceAnalysis(teacherId: string, imageUrl: string) {
     const session = await verifySession()
     if (!session) throw new Error("Unauthorized")

     // AI 관상 분석 실행 (기존 함수 재사용)
     const result = await analyzeFaceImage(imageUrl)

     // DB 저장
     await upsertTeacherFaceAnalysis(teacherId, {
       imageUrl,
       result: result as any,
       status: "complete",
       version: 1,
       analyzedAt: new Date()
     })

     revalidatePath(`/teachers/${teacherId}`)

     return { success: true }
   }
   ```

중요: analyzeFaceImage는 순수 함수로 Student/Teacher 구분 없이 재사용 가능.
