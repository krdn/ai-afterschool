# Plan 17-02: 예약 목록 조회 및 검색

## 목표
예약 목록을 조회하고 학생명, 날짜, 상태로 검색하는 기능을 구현합니다.

## 작업

### Task 1: 데이터베이스 조회 함수 작성
**파일:** `src/lib/db/reservations.ts`

1. `getReservations()` - 예약 목록 조회
   - teacherId 필터링 (현재 세션 선생님)
   - 선택적 필터: studentId, dateFrom, dateTo, status
   - Student, Parent 조인 포함
   - scheduledAt 기준 내림차순 정렬

2. `getReservationById()` - 단일 예약 조회
   - ID로 예약 상세 조회
   - 관련 Student, Parent 정보 포함

### Task 2: Server Action 작성
**파일:** `src/lib/actions/reservations.ts`

1. `getReservationsAction()` - 목록 조회
   - 인증 체크
   - TEACHER 역할 시 자신 예약만 조회
   - 검색 파라미터 전달
   - { success, data? } 형식 응답

2. `getReservationByIdAction()` - 상세 조회
   - 인증 및 권한 체크
   - 예약 데이터 반환

## 검증 기준
- [ ] 선생님이 자신 예약 목록만 조회
- [ ] 학생명 필터로 검색 가능
- [ ] 날짜 범위 필터 동작
- [ ] 상태(SCHEDULED/COMPLETED/CANCELLED/NO_SHOW) 필터 동작
