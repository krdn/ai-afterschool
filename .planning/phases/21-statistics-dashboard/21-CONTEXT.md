# Phase 21: Statistics & Dashboard - Context

**Gathered:** 2026-02-04
**Status:** Ready for planning

<domain>
## Phase Boundary

상담 통계 및 후속 조치 대시보드를 구축합니다. 선생님별/학생별 상담 횟수, 유형별 분포, 월별 추이를 차트로 시각화하고, 후속 조치 목록을 관리할 수 있는 대시보드를 제공합니다.

**포함:**
- 선생님별 월간 상담 횟수 표시
- 학생별 누적 상담 횟수 표시
- 상담 유형별 분포 차트
- 월별 상담 추이 라인/영역 차트
- 오늘/이번 주 후속 조치 목록
- 지연된 후속 조치 하이라이트
- 후속 조치 완료 체크 기능

**미포함:**
- AI 기반 상담 요약 생성 → Phase 22
- 외부 알림 (이메일/SMS) → 별도 Phase

</domain>

<decisions>
## Implementation Decisions

### 통계 시각화
- 월별 상담 추이: 라인 차트 + 영역 차트 토글 전환 지원
- 상담 유형별 분포: 도넛 차트로 전체 대비 비율 표시
- 기본 표시 기간: 최근 6개월
- 선생님별 비교: 테이블 + 미니 바 차트 조합 (순위와 시각적 비교 모두 제공)

### 후속 조치 관리
- 마감일 설정: 상담 기록 작성 시 수동 입력 (DatePicker)
- 지연 표시: 빨간색 배경 + 경고 아이콘으로 강조
- 완료 기록: 체크박스 + 선택적 메모 입력 필드
- 정렬 방식: 마감일 임박순 (긴급한 것 먼저)

### 대시보드 레이아웃
- 요약 카드: 4개 (이번 달 상담, 대기 예약, 지연 후속조치, 완료율)
- 섹션 배치: 2열 그리드 (왼쪽: 통계/차트, 오른쪽: 후속조치 목록)
- 후속 조치 범위: 오늘 + 이번 주
- 모바일: 카드 세로 스택으로 반응형 전환

### 데이터 필터링
- 기간 필터: 프리셋 버튼 (1개월/3개월/6개월/1년) + 사용자 지정
- 선생님 필터: 드롭다운으로 단일 선생님 선택
- 유형 필터: 체크박스로 다중 유형 선택 (학습/진로/생활/기타)
- 내보내기: CSV 다운로드 지원

### Claude's Discretion
- 차트 색상 팔레트 및 스타일링
- 요약 카드 아이콘 선택
- 빈 상태(데이터 없음) UI 디자인
- 로딩 스켈레톤 구현 방식

</decisions>
