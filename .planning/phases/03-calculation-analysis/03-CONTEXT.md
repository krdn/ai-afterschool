<decisions>
## Implementation Decisions

### 생년월일시 입력 규칙
- 출생 시간은 **선택 입력**이며, 미입력 시 **시주는 비움**(일주만 계산)
- 출생 시간 미상은 **미상 상태로 저장**하고 화면에 “시간 미상” 표기
- **양력만 입력**(음력/윤달 입력 없음)
- 시간대는 **KST 고정 + 태양시 보정 포함**

### 사주 결과 출력 구조
- 화면 구성은 **스텝형**: 기본 정보 → 사주 구조 → 해석
- 해석 텍스트는 **자세한 길이**(10문장 이상, 단락 구성)
- 표시 범위는 **천간/지지 + 오행 + 십성 + 대운/세운(요약)**
- 전문용어는 **쉬운 한국어 설명 위주**, 전문용어 최소화

### 성명학 입력 방식
- **한글 + 한자 선택 입력**
- 한글 이름 입력 후 **한자 후보 선택** 방식
- **한자 획수만 사용** (한자 미입력 시 성명학 분석 미제공)
- 성/이름은 **자동 분리(2~4글자 규칙)**

### 저장/갱신 방식
- 분석 실행 **즉시 저장**
- 변경 발생 시 **“재계산 필요” 배지**로 상태 표시
- 결과는 **최신 1개만 유지(덮어쓰기)**
- 상태 표시는 **“최신 분석” / “재계산 필요” 배지** 사용

### Claude's Discretion

</decisions>
