       - MatchMode: 'auto_tag' | 'specific_model'
       - FallbackMode: 'next_priority' | 'any_available' | 'fail'
       - FeatureMappingConfig: DB FeatureMapping 타입
       - FeatureMappingInput: 입력 타입

    5. 레지스트리 관련 타입:
       - RegistryEntry: { provider: ProviderConfig; adapter: Adapter; }
       - ResolutionResult: { provider: ProviderConfig; model: ModelConfig; }

       - Provider 및 연결된 Models 삭제

       - 캐시 조회 → DB 조회

    5. async list(options?: { enabledOnly?: boolean }): Promise<ProviderWithModels[]>
       - 전체 목록 조회

       - Adapter를 통해 모델 목록 동기화

    8. getAdapter(provider: ProviderConfig): BaseAdapter
       - providerType에 맞는 Adapter 반환
