    Form 필드:
    1. 기능 선택 (새로 생성 시):
       - Select 컴포넌트
       - FeatureType 목록 (learning_analysis, face_analysis, etc.)

    3. 태그 기반 모드 (matchMode === 'auto_tag'):
       - 필수 태그 (requiredTags):
         * Multi-select 체크박스: vision, function_calling, json_mode, streaming
       - 제외 태그 (excludedTags):
         * Multi-select 체크박스: free, low, medium, high (costTier)
       - 추가 필터 (선택적):
         * 선호 비용 등급: Select (free, low, medium, high)
         * 선호 품질 등급: Select (fast, balanced, premium)
         * 최소 컨텍스트 윈도우: Number 입력

    4. 직접 지정 모드 (matchMode === 'specific_model'):
       - 제공자 선택: Select (등록된 제공자 목록)
       - 모델 선택: Select (선택된 제공자의 모델 목록)

    5. 우선순위:
       - Number 입력 (1, 2, 3...)
       - 낮을수록 먼저 시도됨

    6. 폴 백 전략:
       - Select: "다음 우선순위 시도" | "사용 가능한任何 모델" | "실패 처리"

    7. 미리보기:
       - 현재 설정으로 ResolutionPreview 표시
       - "새로고침" 버튼으로 결과 업데이트

    8. 액션:
       - "저장" 버튼
       - "취소" 버튼
