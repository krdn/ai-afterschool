# 테스트 계정 정보 (Test Accounts)

개발 및 운영 초기 환경에서 사용할 수 있는 테스트용 계정 목록입니다.
`prisma/seed.ts` 스크립트를 통해 생성됩니다.

## 🔑 계정 목록

| 이름 | 역할 (Role) | 이메일 (ID) | 비밀번호 (PW) | 권한 및 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **관리자** | `DIRECTOR` | **`admin@afterschool.com`** | **`admin1234`** | 시스템 전체 관리, 데이터 삭제 가능 |
| **선생님** | `TEACHER` | **`test@afterschool.com`** | **`test1234`** | 학생 관리, 상담 기록, 성적 입력 등 |

## 💡 참고 사항

* **데이터 초기화**: 데이터베이스 리셋이 필요한 경우 `npx prisma db push --force-reset` 후 `npx prisma db seed`를 실행하면 위 계정들이 다시 생성됩니다.
* **비밀번호**: 모든 비밀번호는 `argon2`로 해싱되어 저장됩니다.
* **보안 주의**: 위 계정은 개발 및 테스트 목적이므로, 운영 환경 배포 시에는 반드시 비밀번호를 변경하거나 새로운 보안 계정을 생성하여 사용하십시오.
